# 🧬 DNA to RNA Transcription

## Biological Insight
Transcription is the first step of gene expression, where a particular segment of DNA is copied into RNA by the enzyme RNA polymerase. Bridging the gap between storage (DNA) and action (Protein), RNA uses **Uracil (U)** instead of **Thymine (T)**.

## Logic & Tools
- **String Manipulation:** Utilized Python's `.replace()` method for efficient character substitution.
- **CLI Ready:** Supports direct file input via `sys.argv`.
- **Validation:** Implemented `.upper()` to handle any non-standard sequence formatting.
